Analysis of Suicide Rates Over Time¶

Many cultures have shamed the act of suicide and those that end their own lives. Whether or not that worked to deter the people of the time is another disscussion, but what we need to focus on now is how suicide rates have changed recently and how we can help those in need.

From here we can see that there has been a progressive rise in suicides over these recent decades. Now this data does not include any information on suicide rates in the year 2020, a landmark year that has changed many peoples lives worldwide. So there will be a large caveat to this analysis that all of this information if before the seismic events caused by the Covid-19 pandemic.

Trying to look at every countries suicide rates like we just did would be time consuming. Lets make a function so that we can get this information easier.

def MakePlots(df,country):
    df = df.loc[df['Location'] == country]
    M_plot = df.loc[df['Dim1'] == 'Male'].plot('Period','FactValueNumeric',kind = 'line',c = 'b',label = 'Males')
    F_plot = df.loc[df['Dim1'] == 'Female'].plot('Period','FactValueNumeric',kind = 'line',ax = M_plot,c = 'pink',label = 'Females')
    A_plot = df.loc[df['Dim1'] == 'Both sexes'].plot('Period','FactValueNumeric',kind = 'line',ax = F_plot,c = 'black',label = 'Both sexes',xticks = years)
    plt.title('Aveage Suicides Over Time in ' + country)
    plt.xlabel('Year')
    plt.ylabel('Suicides per 100,000')
    plt.show()
